# About this Project
# 🚀 Mantime - Beginner Web Project

A simple and responsive personal web project built using **HTML, CSS, and JavaScript**. This project showcases fundamental web development concepts including layout design, navigation, and basic interactivity.

---

## 🌐 Live Demo

🔗 https://shusanadk562.github.io/Mantime/index.html

---

## 📌 Features

* 🧭 **Fixed Navigation Bar**

  * Smooth scrolling between sections
  * Clean and minimal UI

* 🎯 **Hero Section**

  * Background image with blur effect
  * Typing animation using JavaScript

* 👤 **About Section**

  * Simple and centered layout
  * Clean typography

* 📞 **Contact Section**

  * Displays phone, email, and location
  * Structured layout using CSS

* 📱 **Responsive Design**

  * Mobile-friendly layout
  * Media queries for smaller screens

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling, layout, and responsiveness
* **JavaScript** – Basic interactivity and typing effect
